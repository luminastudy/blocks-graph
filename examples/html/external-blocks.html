External Block Icon Feature

GitHub Blocks

External blocks from GitHub repositories display a GitHub icon (black) in the top-right corner.

GitLab Blocks

External blocks from GitLab repositories display a GitLab icon (orange) in the top-right corner.

Other Platforms

External blocks from other platforms (Bitbucket, etc.) display a generic link icon.
